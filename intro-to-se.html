Introduction to Software Engineering

Eager to share my software engineering adventure! This post dives into the first two weeks, exploring what I've learned and the challenges I've faced. Join me as I embark on this exciting journey!

How the Internet works

Internet is a global network of interconnected computers that communicate with each other using standardized protocols. I learnt that the IP addresses and servers are the things someone must know in internet. That is, each device has a unique address allowing information packets to travel and be delivered to a correct destination. Protocols ensure that the packets are sent and received properly and different layers handle specific tasks. what also stood out for me is, Wifi was invented in 1997, then in 2019, 5G was introduced

Introduction to version control

Version control is the process of tracking changes to project files over time. It allows multiple people to collaborate on the same project without overwriting each other's work. Version control is commonly used in software development to manage source code.Git is one of the most widely used version control systems known. This is because it allows easy collaboration among developers working remotly. Versino control also helps someone to revert to previous version in case something goes wrong with the code during development.

Introduction to Html

Html stands for HyperText Markup Language. It provides the structure and organisation to the content that can be displayed on the screen. It is the standard language used to create and design web pages.It consists of differnt elements that are represented by tags enclosed in angle brackets with opening and closing tags. These elements tell the browser to display the content forexample, title, form, image, paragraph. HTML doesn't directly add visual style but it forms the foundation upon which websites are built hence allowing us to create basic structure and content before adding stylings.

Introduction to CSS

CSS stands for Cascading Style Sheets. It is the language responsible for adding style and defining the visual presentation of web pages. It works hand in hand with HTML to structure and style content, allowing designers to control the layout,colors,fonts and other aspects of a website.It controls the layout of multiple web pages all at once. In CSS, each declaration ends with a semi colon(;), CSS includes, external css, internal css and inline css. What stood out for me is, when styling an id selector, write a hash(#) then put the id name,when styling the class selector, write a period(.) then put the class name you want to style.

Git and GitHub

Git is a distributed version control system designed to track changes in source code during software development. It allows developers to collaborate on projects efficiently and maintain history of changes. It also allows us to revisit our previous versions. There are different commands that can be used forexample, gitint that creates and initializes the repository, git commit -m that saves the changes. I also learnt how to connect git repository to cloud, pushing, pulling and also creating sub branches and merging branches. Github is a popular web-based platform built around Git. It serves as a centralized hub for developers to share their code, contribute to open-source projects and collaborate with others in the development community.

Introduction to Bootstrap

Bootstrap is a pre-built toolbox containing ready-made components, saving you time and effort in coding everything from scratch. It is a popular and powerful front-end framework built on top of HTML, CSS, and JavaScript. It provides a collection of pre-designed and customizable components like buttons,forms,navigation bars and more. Its grid system and responsive design utilities enable developers to create layouts that adapt seamlessly to various screen sizes and devices, ensuring a consistent user experience across desktops, tablets, and smartphones. With Bootstrap, developers can streamline their workflow, save time on design and development, and focus more on building robust and visually appealing web solutions.

About

Software engineering serves as the backbone of our digital age, shaping the technology that permeates every aspect of our lives. At its core, software engineering encompasses the systematic application of engineering approaches to the development, maintenance, and evolution of software systems. From the apps on our smartphones to the complex systems running critical infrastructure, software engineers play a vital role in crafting solutions to meet the demands of our interconnected world. In this blog post, we'll delve into the fundamental concepts of software engineering, exploring its principles, methodologies, and the ever-evolving landscape of technologies that drive innovation forward
